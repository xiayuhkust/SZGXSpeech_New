# Text Processor 文本处理工具

## 功能概述
这是一个专门用于处理宗教文本的工具，主要功能包括：
- 文本清理和格式化
- 语法纠正
- 圣经引用标注
- 段落重组

## 处理流程
1. 文本输入
   - 用户通过文本框粘贴输入文本
   - 系统自动处理UTF-8编码

2. 文本分块处理
   - 使用tiktoken进行token计数
   - 按照最大1000字符进行分块
   - 保持段落完整性

3. GPT-4处理
   - 使用专门设计的提示词
   - 保持原文情感不变
   - 清理口语化表达
   - 修正语法错误
   - 标注圣经引用

4. 结果输出
   - 处理完成后通过邮件发送
   - 保持文本格式整洁

## 使用的模型
- GPT-4：用于主要的文本处理
- Tiktoken：用于准确的token计数

## 提示词设计
处理文本时使用的主要提示词规则：
1. 保持原文情感不变
2. 清理冗余词（如：这个、那个、然后、就是等）
3. 合并重复表达
4. 保留特定宗教用语
5. 规范圣经引用格式
